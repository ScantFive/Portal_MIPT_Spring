 # Portal MIPT — бэкенд (Spring)

Бэкенд на Spring Boot для `Portal MIPT`.

## Возможности

- Регистрация, вход, активация и управление профилем пользователя.
- CRUD для объявлений, управление фотографиями, избранное, поддержка аукционов.
- Поиск с фильтрами, постраничной навигацией, историей поисков и подсказками.
- Отзывы и рейтинг продавцов.
- Чаты и обмен сообщениями.
- Операции с кошельком: баланс, резервирование, оплата, отмена и возвраты.
- Публикация событий в Kafka для сервиса уведомлений.

## Недавние дополнения

- Эндпоинты аукционов: выставление ставок и закрытие аукциона.
- API отзывов для обратной связи и оценки продавцов.
- Хранение данных Telegram-пользователей для уведомлений.
- История поиска и подсказки.

## Основные API

- `POST /users`, `GET /users/*` — работа с пользователями
- `GET/POST /v1/advertisements` — объявления и операции над ними
- `POST /v1/advertisements/{id}/auction` — ставки по аукционам
- `GET /search` — поиск и подсказки
- `POST /v1/reviews` — отзывы
- `/chats` — создание и чтение чатов
- `/wallets` — операции кошелька

## Запуск локально

Сначала запустите инфраструктуру, затем приложение:

```bash
docker compose up -d db zookeeper kafka kafka-ui
mvn spring-boot:run
```

## Сборка

```bash
mvn clean package -DskipTests
mvn test
```

## Примечания

- Java 21, Spring Boot 3.5, PostgreSQL и Kafka.
- Конфигурация в `src/main/resources/application.properties` и `application.yaml`.
- SQL-скрипты инициализации в `src/main/resources/db-init/`.
- Бэкенд может раздавать фронтенд из статических ресурсов.
